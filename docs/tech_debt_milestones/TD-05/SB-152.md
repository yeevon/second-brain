# SB-152 — Deterministic local classifier mode for smoke tests

**Branch:** `feature/deterministic-classifier-mode`
**Source:** TD-P3-004 / GitHub #8

---

## Goal

Local smoke tests always burn Gemini API tokens and depend on model behavior. A deterministic classifier mode would make local validation faster, cheaper, and reproducible.

---

## Acceptance criteria

- [ ] Add a local-only deterministic classifier mode.
- [ ] Enable via env var (e.g., `CLASSIFIER_MODE=deterministic`).
- [ ] Deterministic mode returns a fixed, configurable classification result.
- [ ] At least one real Gemini manual/integration validation path is preserved.
- [ ] Tests use deterministic mode by default; real Gemini path is opt-in.

**Done when:** local smoke tests run without Gemini API calls when the deterministic mode env var is set.
