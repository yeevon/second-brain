# 02 · EC2 and n8n setup

## 1. Launch EC2 instance

In AWS Console:

| Setting | Value |
|---------|-------|
| AMI | Ubuntu 24.04 LTS |
| Instance type | t3.small (2GB RAM) |
| Storage | 20GB gp3 |
| Security group | see below |

**Security group inbound rules:**

| Type | Port | Source |
|------|------|--------|
| SSH | 22 | Your IP only |
| Custom TCP | 5678 | 0.0.0.0/0 (n8n webhook) |

> Port 5678 needs to be open to the world so Discord can reach your webhook.
> SSH should be restricted to your IP only.

## 2. Connect and update

```bash
ssh -i your-key.pem ubuntu@your-ec2-ip

sudo apt update && sudo apt upgrade -y
```

## 3. Install Docker

```bash
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker ubuntu
newgrp docker
```

Verify:

```bash
docker --version
```

## 4. Install n8n via Docker

```bash
mkdir -p ~/.n8n

sudo chown -R 1000:1000 ~/.n8n

docker run \
  -d --name n8n \
  --restart unless-stopped \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  -e N8N_HOST=your-ec2-ip \
  -e N8N_PORT=5678 \
  -e N8N_PROTOCOL=http \
  -e WEBHOOK_URL=http://your-ec2-ip:5678/ \
  -e N8N_SECURE_COOKIE=false \
  n8nio/n8n
```

## 5. Verify n8n is running

Open in browser: `http://your-ec2-ip:5678`

On first visit you will see a setup form. Enter your name, email and password — this becomes your n8n owner account. Write these down.

Save your webhook URL:

```init
N8N_WEBHOOK_URL=http://your-ec2-ip:5678
```

## 6. Keep n8n updated

```bash
# To update n8n later
docker pull n8nio/n8n
docker stop n8n && docker rm n8n
# re-run the docker run command above
```
