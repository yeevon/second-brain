# SB-121 — Add weekly review workflow

**Branch:** `feature/weekly-review`

## Purpose

Run a weekly review on EC2 that reads the capture ledger (and optionally the vault) and posts a structured summary plus an AI-generated priorities section to `#brain-digest` or DM. Completion must be based on explicit state changes only — never inferred from vague prose.

## Rules enforced

- `completed_actions_count` counts notes classified with `note_type: done` or `note_type: fix` — not text-pattern matching on body content.
- `created_tasks_count` counts notes classified with `note_type: task`.
- AI output is labeled `⚠️ AI-GENERATED PRIORITIES (review before acting)` and is never written back to the task ledger.
